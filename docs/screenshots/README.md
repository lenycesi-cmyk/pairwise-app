# Screenshots

Dossier pour déposer des captures d'écran servant à discuter d'UI/de bugs avec Claude.

Usage : uploader une image ici (ex. `wealth-bank.png`), puis dire à Claude
« regarde `docs/screenshots/wealth-bank.png` ».

⚠️ Ce repo est **public** : ne pas déposer de captures contenant des données
financières sensibles sans les avoir floutées. Une image commitée reste dans
l'historique Git même après suppression.
